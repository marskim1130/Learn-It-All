用户注册 Demo

演示 POST /auth/register：校验输入、Argon2 密码摘要、邮箱唯一冲突。

API

POST /auth/register
Content-Type: application/json

{
  "email": "alice@example.com",
  "password": "password123"
}

设计思想

  • 公开接口只返回用户公开字段，不返回密码或摘要。
  • 错误使用稳定机器码：VALIDATION_ERROR、EMAIL_ALREADY_EXISTS。
  • 数据库唯一约束兜底并发注册竞态。

可运行 Demo

先启动 API（默认 http://localhost:3001），再在本页提交。

常见错误

  • API 未启动：浏览器显示网络错误。
  • 重复邮箱：状态码 409，错误码 EMAIL_ALREADY_EXISTS。
  • 非法邮箱或过短密码：状态码 400，错误码 VALIDATION_ERROR。
